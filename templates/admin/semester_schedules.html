How it works
  • Enable auto-send and pick a day + time. The system checks every hour and fires exactly once per day when the schedule matches.
  • Use the Department, Semester, and Admission Year filters to target only the students who need reports — saving inbox clutter for everyone else.
  • Each student receives their own subject-wise breakdown. Parents are notified only when a subject drops below {{ config.LOW_ATTENDANCE_THRESHOLD if config else 75 }}%.
  • Use Send Now to trigger an immediate send using the saved filters (great for testing).
