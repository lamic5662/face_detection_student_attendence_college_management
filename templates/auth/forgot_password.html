Recovery Flow

How Recovery Works

Simple reset flow designed to restore access safely while keeping ownership of the password with the user.

1

Request Reset

Submit the email address linked to the SmartAttend account and request a secure recovery link.

2

Receive Email

Open the reset message delivered to the verified inbox connected to the account.

3

Create Password

Use the one-time link to create a new private password directly under the account owner’s control.

4

Sign In Again

Return to the portal and continue securely with the newly updated password.

Security Model

Security Principles

Password recovery follows the same security model used across SmartAttend, with private user control and restricted recovery access.

Protected Token Flow

Reset requests use a time-bound email token instead of exposing passwords or relying on manual sharing.

User-Owned Password

The account owner creates the new password directly, which keeps personal credentials private.

Minimal Exposure

The page does not reveal whether an account exists beyond the secure recovery flow.

Role Coverage

Access Scope

Recovery supports every role in the system while keeping password ownership with the individual user.
